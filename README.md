# 🏦 Loan Default Prediction - End-to-End Data Analysis

This project demonstrates a complete data analysis pipeline, from business requirement gathering to final stakeholder presentation.

## 📂 Project Structure
* **Business Requirement Document (BRD):** Defined project scope and KPIs.
 
* ## 📊 Excel Analysis
I used **Microsoft Excel** for data validation and mapping:
* **Data Mapping:** Extensively used **Lookup functions** to connect and cross-reference different data points.
* **Outlier Analysis:** Performed outlier detection to identify extreme values in the dataset.
* **Pivot Tables:** Created pivot summaries to quickly analyze data distributions.
* **Null Analysis:** Identified and handled missing values (Nulls) to maintain data quality.
  
* ## 🗄️ Database & SQL (MySQL)
I used **MySQL** for data management and extraction:
* **Data Extraction:** Wrote optimized queries to pull data from multiple tables.
* **Data Transformation:** Used SQL functions to clean and format raw data before analysis.
* **Joins & Aggregations:** Performed complex joins to combine customer demographics with loan history.
* **Python (Jupyter Notebook):** Data cleaning and Exploratory Data Analysis (EDA).
* **Power BI Dashboard:** Interactive visualization and reporting.
* **Final Presentation:** Summarized insights for business stakeholders.

---

## 🐍 Python & Statistics Analysis
I performed deep-dive analysis using Python libraries to ensure data quality and statistical significance:

* **Libraries Used:** `Pandas`, `NumPy`, `Matplotlib`, `Seaborn`.
* **Outlier Analysis:** Used **Box Plots** to identify and handle extreme values in Income and Loan amounts.
* **Descriptive Statistics:** Calculated **Mean, Median, Standard Deviation, and Skewness** to understand data distribution.
* **Range Analysis:** Performed min-max checks to identify "Out of Range" values.

---

## 📊 Key Insights
* Identified high-risk customer segments based on credit scores.
* Discovered a correlation between income levels and loan default probability.
* Visualized key KPIs like Default Rate, Total Loan Amount, and Customer Demographics.
